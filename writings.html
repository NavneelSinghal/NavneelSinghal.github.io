Mathematical handouts and expositions

  1. On a rich configuration related to tangent circles in a triangle: This paper explores a rich configuration in a triangle with loads of interesting tangencies. As a bonus, there is also a solution to problem G7 of the IMO 2002 shortist, using the results we develop in the paper.
  2. Kepler’s Laws and the Tautochrone problem: This is an excursion into the seemingly unrelatedness of the two problems as described, and a solution of one using the other.
  3. Diophantine Equations: This is an article on important techniques in Diophantine analysis.
  4. Isogonal Conjugates: This is a short note on isogonality, intended to exhibit its uses in mathematical olympiads.
  5. Geometrical Gems: This is a collection of some beautiful problems in geometry and some problems.
  6. AM-GM and Thermodynamics: This is an article intended to show the AM-GM inequality using thermodynamics.
  7. Lemmas in Olympiad Geometry: This is a collection of useful results in geometry.
  8. Practice Problems in Geometry: This is a collection of some great problems in geometry.
